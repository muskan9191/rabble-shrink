# Rabble-Shrink  
**Online Grocery Marketplace Platform**

## Overview  
Rabble-Shrink is a web application that enables local grocery stores to register, upload and manage their products, and allows customers to browse, discover, and order groceries from nearby stores. The platform connects stores and users in a seamless, efficient way—ideal for neighbourhood grocery commerce.

## Key Features  
- Store registration and login (for grocery shops)  
- Product catalogue management (add/edit/delete items)  
- Browse nearby stores & their inventory (for users)  
- Place orders and manage order status  
- Simple user interface for both store-owners and shoppers  
- Lightweight backend (Python) + front-end (HTML/CSS/JS) → easy to deploy  

## Why This Project Matters  
- Supports local grocery stores to move online quickly  
- Helps customers access nearby inventory instead of generic big-chains  
- Encourages community commerce and shorter supply chains  
- Adaptable for local/regional pickup, delivery, or in-store collection workflows  

## Technology Stack  
- **Backend:** Python (see `main.py`)  
- **Database:** SQL script `grocerystore.sql` for schema setup  
- **Front-end:** HTML, CSS, JavaScript in `templates/` and `static/`  
- **Configuration:** `config.json` for environment settings
